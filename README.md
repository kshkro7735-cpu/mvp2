# 🏡 양구온길 (Yanggu On-gil)
> **군 장병 가족 면회 및 21사단 신병교육대 수료식 방문자를 위한 맞춤형 일정·상점 안내 MVP 서비스**

양구군(특히 21사단 백두산부대)을 찾는 군 장병 가족, 그중에서도 **50~60대 부모님**의 편의를 위해 맞춤형 코스 추천 및 인근 숙소/식당/카페 정보를 손쉽게 탐색할 수 있도록 설계된 모바일 최적화 반응형 정적 웹 애플리케이션입니다.

---

## 💻 기술 스택 & 특징
- **프론트엔드**: Pure HTML5, CSS3, Vanilla JavaScript (ES6)
- **아이콘**: Lucide Icons (CDN 연동)
- **서버 및 데이터베이스**: 없음 (서버리스 정적 웹 사이트 구조)
- **데이터 관리**: `js/data.js` 파일을 데이터베이스 대용으로 사용
- **상태 관리 및 접수 시뮬레이션**: 브라우저의 `localStorage`를 활용하여 체크리스트 상태 유지 및 제휴 신청 데이터 저장
- **UX/UI 디자인**: 모바일 우선(Mobile-First) 디자인, 장년층을 배려한 시인성 높은 큰 폰트 크기 및 터치 영역 제공, 데스크톱 접속 시 가상의 모바일 폰 뷰포트(480px) 중앙 배치

## 📂 프로젝트 구조
```text
mvp2/
├── index.html            # [메인 홈] 21사단 신교대 기본 정보 및 수료식 준비물 체크리스트
├── courses.html          # [추천코스] 사용자 맞춤형 3단계 설문 플래너 및 매칭
├── course-detail.html    # [코스상세] 시간대별 타임라인 동선, 지도 및 전화 다이얼 연동
├── directory.html        # [주변정보] 카테고리 필터링 탭과 실시간 상점 검색 기능
├── contact.html          # [제휴·문의] 양구군 소상공인 입점 신청서 및 팝업 모달 알림
├── css/
│   └── style.css         # 디자인 시스템 정의 (CSS 변수, 하단 탭 바, 컴포넌트 스타일)
└── js/
    ├── data.js           # 15개 업체 상세 데이터, 추천 일정표 원본 데이터
    └── app.js            # UI 주입, 체크리스트, 설문 매칭 로직, 상점 검색/필터, 폼 접수 제어
```

## 🌟 핵심 제공 기능
1. **면회 필수 체크리스트**: 브라우저 새로고침이나 재방문 시에도 사용자가 체크한 항목들의 상태가 그대로 유지됩니다.
2. **맞춤형 동선 추천 플래너**: 동반인, 선호 활동 등에 따라 힐링형/먹방형/감성형 코스 일정을 맞춤 제안합니다.
3. **외부 플랫폼 연동 (길찾기 및 전화)**: 지도 앱 연동 단추 클릭 시 네이버/카카오 지도에서 해당 상점을 찾아주고, 전화 버튼 클릭 시 즉시 통화 연결 화면이 실행됩니다.
4. **소상공인 제휴 접수**: 서버가 없는 한계를 `localStorage` 데이터 기입으로 극복하여 입점 신청 내역을 완벽히 모의 저장합니다.

## 🚀 실행 및 배포 방법
1. 본 레포지토리를 다운로드(또는 클론)합니다.
2. 루트 폴더의 `index.html` 파일을 더블 클릭하면 웹 브라우저에서 바로 즉시 실행됩니다.
3. Netlify, Vercel, GitHub Pages 등을 사용하여 단 몇 초 만에 무료 정적 배포가 가능합니다.
